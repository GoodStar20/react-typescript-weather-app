# Weather App

## Getting Started

### Clone this repo

```bash
$ git clone https://github.com/GoodStar20/react-typescript-weather-app.git
```

### Install dependencies

```bash
$ cd react-typescript-weather-app
$ yarn

$ yarn start
```
